Фото-карусель

Использовал плагин flickity - это красивые, плавные карусели.
  CSS style
<link rel="stylesheet" src="https://unpkg.com/flickity@2/dist/flickity.min.css">
  JavaScript
<script src="https://unpkg.com/flickity@2/dist/flickity.pkgd.js"></script>
Использовал дополнение Flickity fullscreen, чтобы к каруселе добавить кнопку разворачивания на весь экран и чтобы все это функционировало.

Внимание! Версия 2.0.0 имеет критические баги и не работает, так что следует использовать исправленную версию 1.1.1 скрипта. Стиль можно взять от 2.0.0 версии.

Чтобы использовать исправленную версию скрипта, переходим по ссылке и можно скопировать текст, тыкнув по кнопке "Copy raw contents". Создайте файл возле вашего html страницы и назовите его к примеру fullscreen с расширением .js, должен получиться файлик с названием "fullscreen.js", откроте его и вставьте скопированный текст. Осталось только подключить скрипт и стиль:
  CSS style
<link rel="stylesheet" src="https://unpkg.com/flickity-fullscreen@1/fullscreen.css">
  JavaScript
<script src="fullscreen.js"></script>
